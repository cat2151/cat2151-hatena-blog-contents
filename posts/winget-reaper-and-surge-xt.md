## 3行で説明
- オーディオプラグイン、豪華な音色が鳴って素敵ですよね！使うにはDAWが必要なことが多いです
- DAWもオーディオプラグインも、今どきはwingetでインストールできます
- インストールのコマンドと、仮想MIDI鍵盤で音を鳴らすまでの手順を、以下で説明します。10分～20分で音を鳴らせると思います

## インストール

※Windowsが対象です

### REAPER

- [REAPER](https://www.reaper.fm/)は軽量なDAWで、お試し版無料です。

```
winget install Cockos.REAPER
```

- 備考
    - REAPERは `C:\Program Files\REAPER (x64)\reaper.exe` にインストールされます

### Surge XT

- [Surge XT](https://surge-synthesizer.github.io/)はオープンソースのオーディオプラグインで、無料です。

```
winget install "Surge XT"
```

- 備考
    - オーディオプラグインは `C:\Program Files\Common Files\VST3\Surge Synth Team\Surge XT.vst3\Contents\x86_64-win\Surge XT.vst3` にインストールされます

## 起動
- WindowsスタートメニューでREAPERをクリックします。
- REAPERが起動することを確認します。

## Track
- `CTRL + T`を押します。
- Track1 が挿入されることを確認します。

## オーディオプラグイン
- Track1の `FX` ボタンをクリックします。
- `surge` を入力します。
- `vst3 surge XT`（effect が書かれていないもの）を選んで決定します。
- Surge XT画面が表示されることを確認します（いろいろなボタンやスライダーなどがついた画面です）。

## 仮想MIDI鍵盤
- ※REAPERは、PCキーボードとマウスで操作のできる「仮想MIDI鍵盤」を備えています。
- `ALT + B`を押します。
- 仮想MIDI鍵盤が表示されたことを確認します。

## 録音アーム
- ※これをonにしないと、いくら仮想MIDI鍵盤を押しても音が鳴りません
- PCの音量をゼロにします。そうしないとこのあとハウリングします。
    - ※ハウリングとは、PCマイクから拾った音をREAPERがモニタリングし、それをさらにPCマイクからREAPERが拾ってどんどん音が大きくなる現象です
    - ※PCマイクを接続していない場合は、PCマイクの部分は読み飛ばして構いません
- Track1の赤いボタンをクリックします。
- PCマイクの音がTrack1のレベルメーターに表示されたことを確認します。

## input
- ※PCマイクのままでは、ハウリングしてしまいます
- ※PCマイクでなく、仮想MIDI鍵盤をinputにするよう設定しないと、いくら仮想MIDI鍵盤を押しても音が鳴りません
- Track1のレベルメーターを右クリックします。
- input MIDI の、Virtual MIDI Keyboard を選びます。

## 演奏
- 仮想MIDI鍵盤をクリックしたり、PCキーボードのQを押したりします。
- Track1のレベルメーターが変化することを確認します。
- 仮想MIDI鍵盤をクリックしたり、PCキーボードのQを押したりします。
- その音が聴こえるまでPCの音量をあげます。
- 仮想MIDI鍵盤で自由に音が鳴らせることを確認します。
- 音が鳴りました！おめでとうございます！困難な長い道のりを歩んできたあなたの努力は今、報われました。

## 音色ブラウズ
- ※Surge XTには魅力的な音色が多数収録されています。そのバリエーションを楽しみましょう。
- 仮想MIDI鍵盤の`Send all keyboard input to VKB`のチェックをonにします。
    - ※こうしないと、別window操作中に仮想MIDI鍵盤を鳴らせません
- Surge XTのPatchボタンをクリックしながら、QキーやTキーを押します。
- 音色が切り替わりながら音が鳴ることを確認します。
- Surge XTのCategoryボタンをクリックしながら、QキーやTキーを押します。
- 音色が切り替わりながら音が鳴ることを確認します。

## 終わりに
- 音色で遊ぶだけでも楽しいですね！
- DAWは慣れるまで操作の認知負荷が高いアプリです。慣れるまで目的に対して手段を見つけることが困難な、絞り込まれないまま多数表示されるメニューやボタン。モーダルで気がつけば操作に反応しなくなっているGUI。あなたの脳はあっという間に短期記憶を埋め尽くされ悲鳴をあげることでしょう。最初のうちは、疲れないうちに素早く切り上げることをおすすめします。
- 以上です。楽しい音色ライフを！
